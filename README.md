# 🧠 Python OOP Concepts Collection

This repository contains beginner-friendly examples of **Object-Oriented Programming (OOP)** concepts in Python.

It is part of my learning journey where I practice and understand core OOP principles step by step.

---

## 🚀 Topics Covered

* ✅ Classes & Objects
* ✅ Constructors (`__init__`)
* ✅ Encapsulation
* ✅ Inheritance
* ✅ Polymorphism
* ✅ Method Overriding
* ✅ Abstraction (basic level)

## 📂 Project Structure

Each file represents a different concept or example:

```
1.py   → Basic class and object  
2.py   → Constructor example  
3.py   → Encapsulation  
...  
19.py  → Advanced OOP example  
```

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/ssandeepsingapur-netizen/OOP-s-related-code.git
```

2. Navigate to folder:

```
cd OOP-s-related-code
```

3. Run any file:

```
python 1.py
```
